# 3D Football Game 6v6

A multiplayer 3D football game where players on bikes compete in a 6v6 match.

## How to Play

- **W / Up** - Move Forward
- **S / Down** - Move Back
- **A / Left** - Steer Left
- **D / Right** - Steer Right
- **Space** - Jump
- **Shift** - Boost
- **Q / Click** - Pass
- **C** - Toggle Camera
- **M** - Mute Audio

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Tech Stack

- **Client**: Three.js, Vite, TypeScript
- **Server**: Node.js, Socket.IO, TypeScript
- **Physics**: cannon-es
